# Shop SLJ --- Sistema de Loja Virtual (Java Swing)

## Sobre o Projeto

O **Shop SLJ** é um sistema simples de loja virtual desenvolvido em
**Java**, utilizando a biblioteca **Swing** para criação da interface
gráfica.\
O projeto simula o processo de compra, exibição de produtos, carrinho e
finalização do pedido.

------------------------------------------------------------------------

## Estrutura do Projeto

### `ShopSLJ.java`

Contém a lógica principal: - Estrutura de produtos - Controle de
estoque - Controle do carrinho - Cálculo de valores

### `ShopSLJUI.java`

Responsável pela interface gráfica: - Listagem de produtos -
Adicionar/remover itens - Exibição do total - Finalização da compra

------------------------------------------------------------------------

## Tecnologias Utilizadas

-   Java (JDK 8+)
-   Swing (javax.swing)
-   AWT
-   Collections

------------------------------------------------------------------------

## Como Executar

### Via Terminal:

``` bash
javac ShopSLJ.java ShopSLJUI.java
java ShopSLJUI
```

### Via IDE:

-   Abra o projeto na IDE de sua preferência
-   Execute a classe `ShopSLJUI.java`
